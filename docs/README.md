## Technology Stack

- **Framework**: Next.js 16
- **Frontend**: React 19 with TypeScript
- **Styling**: Tailwind CSS v4
- **Components**: shadcn/ui
- **Analytics**: Google Analytics
- **Server**: Vercel

## Development

For local development, ensure you have:
- Node.js 18+
- npm/yarn/pnpm package manager
- Git
