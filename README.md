# Chatura Solutions - Website & Services Platform

A modern, feature-rich Next.js website for Chatura Solutions offering upskilling courses, mock interviews, and educational consultation services.

## 🚀 Quick Start

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the application.

## 🛠 Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Database**: Supabase (optional)
- **Analytics**: Google Analytics
- **CMS**: File-based (JSON for blogs and courses)

## 📁 Key Directories

- `/app` - Next.js app router pages and routes
- `/components` - Reusable React components
- `/lib` - Utility functions and helpers
- `/public` - Static assets and data files
- `/docs` - Complete documentation
- `/public/data/blogs` - Blog content (JSON)
- `/public/data/courses.json` - Courses data
